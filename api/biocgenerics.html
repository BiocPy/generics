biocgenerics.combine module#

biocgenerics.combine.combine(*x: Any)[source]#

biocgenerics.combine.combine(*x: Series)

Combine vector-like objects (1-dimensional arrays).

Custom classes may implement their own combine method.

If the first element in x contains a combine method, the rest of the arguments are passed to that method.

If all elements are 1-dimensional ndarray, we combine them using numpy’s concatenate().

If all elements are either 1-dimensional spmatrix or sparray, these objects are combined using scipy’s hstack.

If all elements are Series objects, they are combined using concat().

For all other scenario’s, all elements are coerced to a list and combined.

Parameters:

x (Any) –

Array of vector-like objects to combine.

All elements of x are expected to be the same class or atleast compatible with each other.

Raises:

TypeError – If any object in the list cannot be coerced to a list.

Returns:

A combined object, typically the same type as the first element in x. If the elements are a mix of dense and sparse objects, a ndarray is returned. A list if one of the objects is a list.

If all elements are 2-dimensional ndarray, we combine them using numpy’s concatenate().
